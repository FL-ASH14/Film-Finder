# Film Finder (Web Application)

The website is up and running. Click here for the link. https://the-film-finder.web.app/ 

A full-stack web application designed to help users discover and source clothing, furniture, and travel destinations featured in their favorite movies and TV shows. This project serves as the original web-based mockup that laid the groundwork for future mobile iterations.

##  Key Features
* **Movie-Specific Discovery:** Users can browse through their favorite films (like *La La Land*) to find exact matches or lookalikes for clothing, set furniture, and real-world filming locations.
* **Full-Stack Architecture:** Distinct separation of concerns with dedicated frontend and backend environments for scalable data management.
* **Interactive Media Integration:** Features embedded audio and interactive elements tailored to specific film pages to enhance the user experience.

## Technical Stack & Architecture
* **Frontend:** (e.g., React.js / HTML5, CSS3, JavaScript)
* **Backend:** (e.g., Node.js / Python)

## Engineering Insights & AI-Powered Workflow

### 1. Prototyping & Layout 
The primary goal of this mockup was to establish a clean, intuitive layout where users could seamlessly transition from browsing a movie to shopping for an item. I mapped out the initial component structures and visual hierarchy before writing the web code.

### 2. AI-Assisted Development
I utilized an **AI-assisted workflow** to speed up the scaffolding of the frontend UI and backend folder structures. 
* **The Process:** I used AI to generate the boilerplate code for the web pages and to help structure the data payloads for the movie items. 
* **The Human Code Review:** AI-generated layouts often require strict manual adjustments to ensure responsive design across different screen sizes. I manually audited the generated code, refined the CSS styling to match the mockup vision, and debugged audio integration features to ensure they fired correctly on specific pages.
